# Legendarium de J.R.R Tolkien
Aplicación web interactiva tipo SPA (Single Page Application) que muestra información detallada sobre el universo de El Señor de los Anillos y otras obras del autor, cargando el contenido de forma dinámica.  
  


Desarrollo completo de la lógica en JavaScript sin frameworks, utilizando peticiones asíncronas (fetch, async/await) para consumir y renderizar datos estructurados desde un archivo data.json.  

Manipulación avanzada del DOM para generar dinámicamente listas, tablas y secciones de contenido según la navegación del usuario.  

Diseño de interfaz con CSS3, implementando un menú lateral interactivo (sidebar), variables CSS para el manejo de temas y animaciones/transiciones fluidas.
